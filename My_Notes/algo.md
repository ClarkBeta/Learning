## 03|复杂度分析（上）：如何分析、统计算法的执行效率和资源？
复杂度分析是精髓
### 1.大O表示法
所有代码的执行时间T(n)与每行代码的执行次数n成正比.
```T(n)=O(f(n))```
T(n)表示时间,n表示数据规模,f(n)表示每行代码执行次数的总和
这就是**大O时间复杂度表达法**,表示代码执行时间岁数据规模增长的**变化趋势**,也叫**渐进时间复杂度**,即时间复杂度.
n很大是,可以忽略低阶,常量,系数
### 2.时间复杂度分析
#### 1.只关注执行最多的代码
```java
int cal(int n){
    int sum=0;
    int i=1;
    for (;i<=n;++i){
        sum=sum+i;
    }
    return sum;
}
```
只需要关注循环次数最多的第4,5行代码
